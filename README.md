
# VolleyStep

## English

VolleyStep is a fullstack ecommerce platform for volleyball shoes.

The project is being built as a portfolio project for a Fresher/Junior Fullstack Developer position.

### Planned technology stack

- Monorepo: pnpm workspace
- Frontend: React, Vite, TypeScript
- Backend: Node.js, Express, TypeScript
- Validation: Zod
- Database: MongoDB Atlas, Mongoose
